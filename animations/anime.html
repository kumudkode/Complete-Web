Animation Library

Explore the powerful techniques available in Anime.js

Basic Properties

Simple animations of CSS properties like position, size, and color.

anime({
  targets: '.element',
  translateX: 250,
  scale: 2,
  duration: 1000
});

Timeline Sequencing

Control complex sequences of animations with precise timing.

const timeline = anime.timeline();
timeline
  .add({ targets: '.el', translateX: 250 })
  .add({ targets: '.el', translateY: 50 });

Staggered Animations

Create cascading effects with delays between elements.

anime({
  targets: '.element',
  scale: [0, 1],
  delay: anime.stagger(100)
});

Shape Morphing

Smoothly transform between different shapes.

anime({
  targets: '.shape',
  borderRadius: ['0%', '50%'],
  easing: 'easeInOutQuad'
});

Keyframes

Define multiple steps in a single animation.

anime({
  targets: '.element',
  translateX: [
    { value: 250, duration: 1000 },
    { value: 0, duration: 1000 }
  ]
});

Physics-Based

Create spring, bounce, and elastic movements.

anime({
  targets: '.element',
  translateX: 250,
  easing: 'spring(1, 80, 10, 0)'
});
